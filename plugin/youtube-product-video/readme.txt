Embed a YouTube video on WooCommerce single product pages by adding a URL to each product.

== Description ==

YouTube Product Video adds a "YouTube Video URL" field to every WooCommerce product. When a URL is saved, a responsive 16:9 embed (via youtube-nocookie.com) is displayed above the product summary on the single product page.

= Features =
* Per-product YouTube URL field in the product sidebar
* Supports youtube.com/watch?v= and youtu.be/ link formats
* Privacy-friendly: uses youtube-nocookie.com
* Responsive 16:9 embed, loads only on single product pages

== Installation ==

1. Upload the `youtube-product-video` folder to `/wp-content/plugins/`.
2. Activate the plugin in **Plugins > Installed Plugins**.
3. Edit any product and paste a YouTube URL into the **YouTube Video URL** field in the sidebar.
4. Save the product. The video will appear above the product summary on the front end.
