Let customers enter personalized text (names, messages, engravings) directly on WooCommerce product pages.

== Description ==

**Product Custom Text for WooCommerce** adds a fully configurable text-input field to any product page so your customers can provide personalized text — perfect for engraved jewelry, custom print products, embroidered apparel, and more.

= Key Features =

* ✅ Enable per-product from the product edit screen (sidebar meta box)
* ✅ Choose between a **textarea** (multi-line) or **text** (single-line) field
* ✅ Customizable **label** and **placeholder** text
* ✅ Set a **maximum character limit** (with a live counter on the front-end)
* ✅ Mark the field as **required** — prevents add-to-cart if left blank
* ✅ Personalization text is displayed in the **cart**, **checkout**, **order confirmation**, and **admin order screen**
* ✅ Text is saved to each order line item so it persists in emails and reports
* ✅ Compatible with WooCommerce **High-Performance Order Storage (HPOS)**
* ✅ No external dependencies, no tracking

= How It Works =

1. Open any product in **WooCommerce → Products**.
2. In the sidebar find **Custom Text Field** and check *Enable custom text field on this product*.
3. Fill in the label, placeholder, field type, max characters, and whether the field is required.
4. Save the product.
5. Customers see the field on the product page and their text follows the product through cart → checkout → order.

== Installation ==

1. Upload the `product-custom-text` folder to `/wp-content/plugins/`.
2. Activate the plugin through **Plugins → Installed Plugins**.
3. WooCommerce must be installed and active.
